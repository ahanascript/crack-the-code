# 🔐 Crack The Code

A number guessing game where you crack a secret code. Built with HTML, CSS, JavaScript, and Supabase.

## Features

- Login with username
- Choose digits (2–6) and toggle repeats
- Guess the code, get position‑wise feedback
- Leaderboard (top 10 players)
- Anonymous feedback
- Mobile responsive

## Tech Stack

- HTML, CSS, JavaScript
- Supabase (PostgreSQL)
- Vercel (deployment)

## Live Demo

https://crack-the-code-eight.vercel.app/

## Setup

1. Clone the repo
2. Add your Supabase URL and anon key in `script.js`
3. Deploy to Vercel
